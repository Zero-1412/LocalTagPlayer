# Local Tag Player Codex 技能 v2

安装到项目根目录：

```text
<project-root>\.agents\skills\
```

在 `<project-root>` 解压此包，让 `.agents/skills` 目录直接位于仓库根目录下。

如果技能没有出现，请重启 Codex 或重新打开项目。

推荐使用：

```text
$ltp-task-router
$ltp-small-fix
$ltp-log-triage
$ltp-session-handoff
$ltp-media-library-tag-ui
```

优先使用 repo-scoped 安装。除非希望 Local Tag Player 规则影响所有项目，否则不要全局安装这些技能。
