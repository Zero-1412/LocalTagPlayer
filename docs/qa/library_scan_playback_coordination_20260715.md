## 本轮协调策略

1. 目录发现只显示已找到数量，不伪造百分比。
2. 候选确定后，fingerprint 显示处理数/总数、百分比、速度和 ETA。
3. 播放开始前暂停 sidecar 文件读取，播放器退出后原位继续。
4. 用户手动暂停优先，播放器退出不能擅自恢复。
5. Application 合并每 256 项让出 UI isolate，避免提交阶段长时间独占输入。

debug 运行会继续写入 `%TEMP%\local_tag_player_scan_ui_diagnostics.jsonl`，其中 `scanPhases` 只保存阶段、数量和耗时，不保存路径、标题或标签。下一轮重启后的真实冷盘操作应记录设备类型、阶段耗时、播放进入耗时和播放期间窗口响应状态。
